# Full i18n epic

CODEX UI je podrazumijevano na **crnogorskom** (`me`). Engleski (`en`) je sekundarni jezik za testiranje i buduće korisnike.

## Stack

- **Bez vanjskog paketa** — lagani modul u `src/i18n/`
- Katalozi: `src/i18n/catalogs/me.ts`, `en.ts`
- React: `I18nProvider` + `useI18n()` → `{ locale, setLocale, t }`
- Persistencija: `AppSettings.locale` (localStorage mirror + SQLite `kv`)

## Korištenje

```tsx
import { useI18n } from "@/i18n";

function MyWidget() {
  const { t, locale, setLocale } = useI18n();
  return <button>{t("common.back")}</button>;
}
```

Novi ključevi: dodaj u **oba** kataloga (`me.ts` i `en.ts`). TypeScript `TranslationKey` se automatski izvodi iz `me` kataloga.
